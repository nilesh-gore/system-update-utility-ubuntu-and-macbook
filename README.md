# Shell Scripts

## Create a Script
```
cd ~/your-project
touch update_util.sh
nano update_util.sh
```

### Example content:
```
#!/bin/bash
echo "Hello, world!"
date
```

### Make Executable
```
chmod +x update_util.sh
```

### Run Script
```
sudo ./update_util.sh
```
### or
```
bash update_util.sh
```
